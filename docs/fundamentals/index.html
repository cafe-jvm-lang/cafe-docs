Function#

In CAFE, functions can be declared using keyword func.

Syntax

func name([param[, param,[..., param]]]) {
[statements]
}

name The function name.

param (Optional) The argument names to be passed to the function.

statements The statements which comprise the body of the function.

Example

func areaOfTriangle(base, height){
return 0.5 * base * height;
}
cmd.print(areaOfTriangle(5, 10)); # 25
